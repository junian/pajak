Keterangan

Norma dalam persen (%) dibagi 3 macam kategori

  • 10 (sepuluh) ibukota propinsi yaitu Medan, Palembang, Jakarta, Bandung, Semarang, Surabaya, Denpasar, Manado, Makassar, dan Pontianak
  • ibukota propinsi lainnya
  • daerah lainnya

Jika menemukan bug, ada data yang salah/tidak lengkap, atau ingin mempercantik UI, silahkan kirim Pull Request ke GitHub
